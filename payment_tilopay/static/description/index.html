  1. Go to Menu > Apps and search Tilopay.
  2. Go to Website > Payment Providers and search Tilopay and Activate.
  3. Go to your Tilopay Account > Checkout and copy the API Key, Username, and Password for the Tilopay API.
  4. Paste the provider credentials from Tilopay Checkout and enable the payment method.
  5. In Checkout enter the required information like First Name, Last Name, etc.
  6. Test the payment and enjoy using Tilopay!.
